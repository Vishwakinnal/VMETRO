# 🚇 VMETRO - Smart Digital Ticket System

A professional Python-based desktop application for **Namma Metro Bengaluru**. 
It calculates exact track distances and generates virtual QR tickets that 
pop up on mobile devices.

### 🌟 Key Features
- **Exact Distance Engine:** Uses BMRCL chainage data for precise KM calculation.
- **Official Fare Slabs:** Matches Bangalore Metro's ₹10-₹95 pricing.
- **Interchange Logic:** Handles Majestic and RV Road transfers automatically.
- **Virtual QR:** Generates iCalendar-formatted tokens for mobile pop-ups.

### 🛠️ Installation
1. Clone the repo: `git clone https://github.com/vishwakinnal/VMETRO.git`
2. Install dependencies: `pip install pillow qrcode`
3. Run the app: `python metro_app.py`
